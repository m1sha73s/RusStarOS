# RusStarOS (РУССКИЙ)
RusStarOS - Это минимальная (пробный проект) операционная система русского и английского издания
_____________________________________________________________________
Требования
CPU: Intel Pentium III и выше или AMD K7 и выше
RAM: МИНИМУМ 16MB
VideoCard 32МБ памяти, 256 цветов и разрешения 1024х768
Disk Space: не менее 4MB в формате FAT
